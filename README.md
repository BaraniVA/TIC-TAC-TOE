Tic Tac Toe Game:
This is a simple Tic Tac Toe game built using HTML, CSS, and JavaScript. The game allows two players to play Tic Tac Toe on a 3x3 board. The game checks for a winner after each round and displays an alert message when a winner is found. The game also checks for a tie and restarts the game when the board is full. The game can also be reset by clicking on the reset button.

Features:
3x3 board
Two player gameplay
Winner notification
Tie notification
Reset button

Technologies:
HTML
CSS
JavaScript
